# Contributing to AccessLM

Thanks for your interest in improving AccessLM. This project is open to all contributors.

## Quick Start
- Fork the repo and create your branch from `main`.
- Keep changes focused and small.
- Add or update tests where appropriate.
- Ensure `npm run lint` and `npm run test` (if present) pass.

## Development Setup
1. Install Node.js (18+ recommended).
2. Install dependencies: `npm install`
3. Start dev: `npm run dev`

## Submitting Changes
- Open a Pull Request with a clear description and screenshots for UI changes.
- Link any related issues.
- Avoid breaking changes unless discussed first.

## Reporting Bugs
Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Screenshots / logs
- OS and version

## Security Issues
Please report security issues to security@accesslm.org.

Thanks for helping make AccessLM better.
