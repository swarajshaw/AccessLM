# Security Policy

We take security seriously. If you discover a vulnerability, please report it responsibly.

## How to Report

Email: security@accesslm.org  
PGP Key: https://github.com/accesslm/accesslm/raw/main/accesslm.asc

## What to Include

- Description of the issue
- Steps to reproduce
- Impact
- Suggested fix (optional)

## What We Do

- Acknowledge within 48 hours
- Fix within 7 days for critical issues
- Publish a public advisory if needed

## We Do Not Accept

- Reports about third-party services (Hugging Face, etc.)
- Social engineering
- Denial-of-service attacks

Thank you for helping keep AccessLM safe.
